# Desktop Pet

一个基于 Python `tkinter` 的桌面宠物示例项目。

## 当前已实现

- 首次启动时弹出宠物选择界面
- 可选择 `小兔`、`小猫`、`小狗`
- 选择结果会保存到 `data/settings.json`
- 主界面支持打开“设置”面板
- 可在设置面板中重新选择宠物
- 可恢复默认宠物
- 可控制“下次启动直接继承当前宠物”或“下次启动重新进入选择界面”
- 鼠标在窗口外时，宠物会朝鼠标方向移动
- 鼠标进入窗口后，宠物眼神会跟随鼠标

## 当前交互说明

- 第一次启动时，会弹出宠物选择界面
- 后续启动默认继承上一次确认保存的宠物
- 如果用户想更换宠物，可在主界面点击“设置”
- 在设置面板里可以重新选择、恢复默认，或者清除启动记忆
- 清除启动记忆后，当前显示的宠物不会立即变化，但下次启动会重新进入选择界面

## 运行方式

确保本机已安装 Python 3，然后在项目根目录执行：

```bash
python main.py
```
